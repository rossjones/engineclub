======================================
Engineclub documentation
======================================

.. rubric:: All about Engineclub.

Engineclub was started in January 2010 as part of the ALISS project for the National Health Service in Scotland.

See the `ALISS Project blog`_ and follow `@ALISSproject`_ on Twitter.

.. _ALISS Project blog: http://www.aliss.scot.nhs.uk/
.. _@ALISSproject: http://twitter.com/alissproject

Contents:

There are four applications:

* aliss - the home page for aliss.engineclub.org which will host information, demos etc
* depot - storage of resource data, and services API, using a mix of mongodb and postgresql (for geoDjango location stuff)
* enginecab - admin app for the engineclub
* firebox - for data collection code
